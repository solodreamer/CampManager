# CampManager Front-End Project

## Project Information
- **Project Name**: CampManager
- **Framework**: React 18
- **Routing**: React Router v6
- **UI Library**: Ant Design
- **Language**: JavaScript
- **Package Manager**: npm

### Directory Structure
src/
├── module/              # Reusable UI components
├── pages/               # Page routes  
│   └── user/            # User pages
├── App.js               # Router configuration file
├── index.css            # Global styles
├── api.js               # Centralized API calls with dynamic authentication                         
└── .env                 # API domain configuration file
```

## Code Standards

### Coding Standards
- **Architecture Principles**: Clean Code, SOLID principles
- **Rules of React**: Hooks should only be called at the top level (not inside conditions, loops, or nested functions), Components and Hooks should be pure, avoid unexpected side effects, use Strict Mode to help identify potential issues
- **Naming Rules**: Clear, concise, rhythmic
- **Comments**: Brief, like song lyrics, not a novel
- **Styling**: Use className instead of inline styles for better reusability and maintainability

## How to Start 
# Install dependencies
npm install

# Start the project
npm run start

# Build the project
npm run build
```
